# Sistema de Confeitaria

Projeto acadêmico de uma aplicação web para controle de clientes e pedidos de uma confeitaria.

## Tecnologias utilizadas

- Next.js
- React
- Tailwind CSS
- Node.js
- Express
- MySQL
- Git e GitHub

## Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Edição de clientes
- Exclusão de clientes
- Cadastro de pedidos
- Listagem de pedidos
- Edição de pedidos
- Exclusão de pedidos

## Banco de dados

O script de criação do banco está localizado em:

database/banco.sql
